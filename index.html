My final work

For my final work I have created web-site for a law firm, for which I used HTML, CSS, Bootstrap, JS, php and SQL. Beside basic information about the firm, this website contains CMS which allows administrator (lawyer) to write posts on the blog, to manipulate with posts (editing, removing), to add new administrators, to change personal information etc.

This website is not online yet, but it can be presented on a request.

This web-site also contains comment section for every blog post. When visitor submits a comment on the post, the comment is not visible immediately on blog page. The comment is visible on the admin panel where admin can approve, disapprove or delete the comment. On blog page are shown only comments which are approved by the admin.

Also everything that visitor enters in fields goes through a security check. First, it is checked if there is any unallowed character. If there is an unallowed character the error message will be shown. In case everything is ok, the data will be written using prepared statements.

Web-site also supports two languages. First is Serbian and the second is English. Visitors can select on wich language the content will be shown by clicking on the button.
